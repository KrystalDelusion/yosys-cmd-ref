=================================
copy - copy modules in the design
=================================

.. only:: html

    :code:`yosys> help copy`
    ----------------------------------------------------------------------------


    :code:`copy old_name new_name` ::

        Copy the specified module. Note that selection patterns are not supported
        by this command.

.. only:: latex

    ::

        
            copy old_name new_name
        
        Copy the specified module. Note that selection patterns are not supported
        by this command.
