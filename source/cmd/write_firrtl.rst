============================================
write_firrtl - write design to a FIRRTL file
============================================

.. only:: html

    :code:`yosys> help write_firrtl`
    ----------------------------------------------------------------------------


    :code:`write_firrtl [options] [filename]` ::

        Write a FIRRTL netlist of the current design.
        The following commands are executed by this command:
                pmuxtree
                bmuxmap
                demuxmap
