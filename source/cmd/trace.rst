=======================================
trace - redirect command output to file
=======================================

.. only:: html

    :code:`yosys> help trace`
    ----------------------------------------------------------------------------


    :code:`trace cmd` ::

        Execute the specified command, logging all changes the command performs on
        the design in real time.

.. only:: latex

    ::

        
            trace cmd
        
        Execute the specified command, logging all changes the command performs on
        the design in real time.
