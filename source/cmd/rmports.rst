=================================================
rmports - remove module ports with no connections
=================================================

.. only:: html

    :code:`yosys> help rmports`
    ----------------------------------------------------------------------------


    :code:`rmports [selection]` ::

        This pass identifies ports in the selected modules which are not used or
        driven and removes them.

.. only:: latex

    ::

        
            rmports [selection]
        
        This pass identifies ports in the selected modules which are not used or
        driven and removes them.
