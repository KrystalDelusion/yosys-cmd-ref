==================================================
echo - turning echoing back of commands on and off
==================================================

.. only:: html

    :code:`yosys> help echo`
    ----------------------------------------------------------------------------


    :code:`echo on` ::

        Print all commands to log before executing them.



    :code:`echo off` ::

        Do not print all commands to log before executing them. (default)

.. only:: latex

    ::

        
            echo on
        
        Print all commands to log before executing them.
        
        
            echo off
        
        Do not print all commands to log before executing them. (default)
