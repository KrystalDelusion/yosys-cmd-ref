====================================
alumacc - extract ALU and MACC cells
====================================

.. only:: html

    :code:`yosys> help alumacc`
    ----------------------------------------------------------------------------


    :code:`alumacc [selection]` ::

        This pass translates arithmetic operations like $add, $mul, $lt, etc. to $alu
        and $macc cells.
