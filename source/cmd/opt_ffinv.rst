======================================
opt_ffinv - push inverters through FFs
======================================

.. only:: html

    :code:`yosys> help opt_ffinv`
    ----------------------------------------------------------------------------


    :code:`opt_ffinv [selection]` ::

        This pass pushes inverters to the other side of a FF when they can be merged
        into LUTs on the other side.

.. only:: latex

    ::

        
            opt_ffinv [selection]
